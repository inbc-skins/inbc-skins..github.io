最終更新：2023-09-18

■使い方

[Board settings] -> [Board textures] -> [Custom Board] でフォルダを指定して再ログインして下さい

・適用には再ログインが必要な箇所とそうでないところがあります
　ベアオフトレイ背景とダイス変更には再ログインが要ります（たぶん）
　何かしらが変更されない場合は再ログインして下さい

・ダイスの変更は一度で成功しない事があるかもしれません（一部の目だけ変わらない）
　その場合は再度フォルダ指定＆再ログインを繰り返してみて下さい



■Custom Board リセット

[Board textures] 右上のゴミ箱アイコンを押して再ログインして下さい

・リセットによりボードデザインが初期化されますが、チェッカーのボーダー設定は据え置きなようです
